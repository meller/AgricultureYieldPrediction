ğŸ“ SUPPORT REFERENCE
================================================================================

If Streamlit won't start:
- Check Python version: python3 --version (need 3.8+)
- Check venv: source venv/bin/activate
- Check dependencies: pip list | grep -E "streamlit|pandas|plotly"

If charts don't render:
- Refresh browser: Ctrl+F5
- Check browser console for errors: F12
- Try Chrome or Firefox (better compatibility)

If data file missing:
- Verify location: ls *.csv
- Check current directory: pwd
- Run from project root, not subdirectory

If port conflicts:
- Use alternate port: streamlit run app.py --server.port 8502
- Or find what's using 8501: lsof -i :8501
